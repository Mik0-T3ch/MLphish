# MLphish

MLpishish es un modelo experimental de *Machine Learning* diseñado para detectar enlaces de *phishing* o potencialmente maliciosos.  
El objetivo principal es mejorar la seguridad del usuario mediante la identificación temprana de links sospechosos.

---

## 🚀 Estado del proyecto

Actualmente el proyecto se encuentra en fase de diseño y experimentación.  
Se está trabajando en la definición del modelo de aprendizaje automático y en la recolección de datos para el entrenamiento.

---

## 🧠 Objetivo

Desarrollar un modelo capaz de:

- Analizar y clasificar enlaces web.
- Identificar patrones comunes en links de *phishing*.
- Alertar al usuario sobre posibles riesgos en tiempo real.

---

## 🧩 Tecnologías previstas

- Lenguaje principal: Python  
- Frameworks de Machine Learning: a confirmar  
- Posibles expansiones:  
  - Interfaz web para analizar enlaces desde el navegador.  
  - Bot de Telegram para detección de enlaces sospechosos en tiempo real.
